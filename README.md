# PromptCanvas (디자인 프롬프트 빌더)

카드뉴스·유튜브 썸네일·상세페이지·배너·인스타 포스트 등
**디자인용 AI 프롬프트**를 옵션 조합으로 뽑아주는 정적 웹앱.

## 쓰는 법

1. `index.html`을 브라우저로 더블클릭해서 열기 (설치·로그인 불필요)
2. 왼쪽에서 디자인 유형·분위기·컬러·레이아웃 등 옵션 선택
3. 오른쪽에 생성된 프롬프트를 **복사**
4. Claude / Midjourney / Canva 등에 붙여넣어 디자인 완성

## 방향 (하이브리드)

```
웹앱에서 프롬프트 뽑기 → Claude로 실제 디자인 생성 → Adobe Express/Canva에서 직접 수정
```

- 비용·서버·API키 없이 "실제 결과물"까지.
- 자세한 결정 배경: [`CLAUDE.md`](./CLAUDE.md)
- 이어서 작업하기: [`다음에-이어서.md`](./다음에-이어서.md)

## 폴더

- `index.html` — 메인 작업본
- `docs/` — 기획서(PRD), 기획 메모
- `assets/preview/` — 디자인 결과 미리보기
- `assets/archive/` — 이전 버전 백업
- `assets/reference/` — 참고 이미지
