프롬프트만이 아니라
결과물 미리보기까지

이전 MVP를 한 단계 올려서, 이제는 사용자가 조합형 프롬프트를 만드는 것에서 끝나지 않고 실제 카드뉴스/썸네일/상세페이지 느낌의 결과물 미리보기를 바로 볼 수 있게 구성했습니다. 즉, 기획 → 프롬프트 생성 → 복사 → 시안 확인까지 한 화면에서 흐름이 이어집니다.

프롬프트 생성 실시간 결과물 미리보기 프리셋 저장 TXT 내보내기 브랜드 템플릿 확장 가능
Studio V2실제 서비스 방향으로 확장한 프로토타입. 선택값이 프롬프트와 결과물 UI에 동시에 반영됩니다.
라이브 프리뷰카드뉴스, 썸네일, 배너, 상세페이지 느낌을 HTML 기반으로 즉시 시각화합니다.
다음 확장 가능원하면 이후 단계에서 실제 AI 이미지 생성 연결도 붙일 수 있습니다.
카드뉴스 유튜브 썸네일 쇼핑 상세페이지 광고 배너 인스타 포스트

1. 템플릿 선택 + 조건 입력

프리셋으로 시작하거나 직접 옵션을 조합해도 됩니다. 저장된 값은 브라우저에 유지됩니다.

2. 생성 프롬프트

상세 프롬프트
왼쪽에서 옵션을 입력한 뒤 “프롬프트 + 결과물 생성”을 눌러주세요.

3. 결과물 미리보기

AI 생성 연결 전 단계에서도 실제 화면 분위기를 확인할 수 있도록 규칙 기반 시안을 즉시 렌더링합니다.

라이브 프리뷰

지금 상태에서 가능한 실제 결과물

  • 프롬프트 즉시 생성
  • 시안형 카드/배너/썸네일 HTML 미리보기
  • 저장/재호출 가능한 프리셋
  • 프롬프트 TXT 내보내기
